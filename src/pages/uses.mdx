# Uses

## Environment

- [Arch](https://archlinux.org) - Lightweight linux distribution with up-to-date packages repository
- [Brave](https://brave.com) - Privacy focused Chrome alternative
- [Fish](https://fishshell.com) - User-friendly feature-rich modern shell
- [Gnome](https://www.gnome.org) - Desktop environment with Wayland support
- [JetBrains](https://www.jetbrains.com/lp/mono) - Monospace font with ligatures support
- [Kitty](https://sw.kovidgoyal.net/kitty) - GPU accelerated terminal emulator with good ligatures support
- [Neovim](https://neovim.io) - Vim-based text editor
- [Tmux](https://github.com/tmux/tmux) - Multiplexer to manage terminal sessions

## CLI

- [FZF](https://github.com/junegunn/fzf) - Fuzzy finder for shell scripts and text editor search
- [Ripgrep](https://github.com/BurntSushi/ripgrep) - Better grep and also works with FZF
- [Stow](https://www.gnu.org/software/stow) - Dotfiles management
- [Vifm](https://vifm.info) - Vim-based file manager
- [Volta](https://volta.sh) - JavaScript tools management
