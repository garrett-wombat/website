# Uses

## Environment

- [Arch](https://archlinux.org) - Lightweight Linux distribution
- [Brave](https://brave.com) - Privacy focused browser
- [Fish](https://fishshell.com) - User friendly interactive shell
- [Gnome](https://www.gnome.org) - Desktop environment - Wayland
- [JetBrains](https://www.jetbrains.com/lp/mono) - Font with ligatures support
- [Kitty](https://sw.kovidgoyal.net/kitty) - GPU accelerated terminal emulator
- [Neovim](https://neovim.io) - Vim-based text editor
- [Tmux](https://github.com/tmux/tmux) - Terminal sessions management

## CLI

- [FZF](https://github.com/junegunn/fzf) - General purpose fuzzy finder
- [Ripgrep](https://github.com/BurntSushi/ripgrep) - Better grep
- [Stow](https://www.gnu.org/software/stow) - Dotfiles management
- [Vifm](https://vifm.info) - Vim-based file manager
- [Volta](https://volta.sh) - JavaScript tool manager
