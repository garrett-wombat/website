# Uses

## Environment

- [Arch Linux](https://archlinux.org) - Lightweight distribution with up-to-date packages
- [Brave](https://brave.com) - Privacy focused Chrome alternative
- [Fish Shell](https://fishshell.com) - User friendly shell with useful out of the box features
- [Gnome](https://www.gnome.org) - Desktop environment with Wayland support
- [JetBrains Mono](https://www.jetbrains.com/lp/mono) - Monospace font for programming with ligatures support
- [Kitty Terminal](https://sw.kovidgoyal.net/kitty) - GPU accelerated terminal emulator with ligatures support
- [Neovim](https://neovim.io) - Vim-based text editor
- [Tmux](https://github.com/tmux/tmux) - Multiplexer to manage terminal sessions

## CLI

- [FZF](https://github.com/junegunn/fzf) - Fuzzy finder for the shell and the text editor
- [Ripgrep](https://github.com/BurntSushi/ripgrep) - Better grep and works well with FZF
- [Stow](https://www.gnu.org/software/stow) - General configuration files (dotfiles) management
- [Vifm](https://vifm.info) - Vim-based file manager
- [Volta](https://volta.sh) - JavaScript tools management
